# 🌟 lunus AI 앱 빌더 - 전체 기능 가이드

## 🎯 핵심 기능

### 1️⃣ 5가지 업종 템플릿 시스템

#### 🍽️ 요식업 - 메뉴 추천 AI
- **데이터**: 메뉴명, 가격, 재료, 알러지, 맵기, 카테고리
- **기능**: 고객의 알러지와 선호도에 맞는 메뉴 추천
- **예시 데이터**: 10개 한식 메뉴 자동 로드

#### 🏠 부동산 - 매물 추천 AI
- **데이터**: 매물명, 위치, 가격, 평수, 방/욕실 개수
- **기능**: 예산과 선호 지역에 맞는 매물 추천
- **예시 데이터**: 5개 아파트 매물 자동 로드

#### 🏥 의료 - 진료과 추천 AI
- **데이터**: 진료과명, 담당의, 전문분야, 증상
- **기능**: 증상에 맞는 진료과 추천
- **예시 데이터**: 5개 진료과 정보 자동 로드

#### 🛍️ 쇼핑몰 - 상품 추천 AI
- **데이터**: 상품명, 가격, 카테고리, 브랜드
- **기능**: 스타일과 예산에 맞는 상품 추천
- **예시 데이터**: 5개 상품 자동 로드

#### ✈️ 여행 - 여행지 추천 AI
- **데이터**: 여행지, 국가, 시즌, 액티비티
- **기능**: 여행 스타일에 맞는 여행지 추천
- **예시 데이터**: 5개 여행지 정보 자동 로드

---

### 2️⃣ 각 템플릿별 3가지 UI 테마

모든 업종에서 선택 가능:

#### 🎨 모던
- 세련되고 트렌디한 디자인
- 다크/컬러풀 조합
- 그라데이션 효과

#### 🏛️ 클래식
- 전통적이고 고급스러운
- 따뜻한 색조
- 우아한 세리프 폰트

#### ⚡ 미니멀
- 깔끔하고 심플한
- 화이트/그레이 조합
- 깔끔한 선과 간격

**총 15가지 조합 (5 업종 × 3 테마)**

### 3️⃣ 데이터 입력 시스템

#### 📝 텍스트 입력 (기본)
- **자동 샘플 데이터**: 템플릿 선택 시 예시 데이터 자동 입력
- **수정 가능**: 자유롭게 추가/수정/삭제
- **간편 형식**: 한 줄에 하나씩, 쉼표로 구분

```
예시 (요식업):
김치찌개, 12000, 얼큰한 국물 요리
비빔밥, 13000, 건강한 한식
된장찌개, 10000, 구수한 맛
```

#### 📁 파일 업로드
- **Excel**: .xlsx, .xls
- **CSV**: 쉼표/탭 구분
- **JSON**: 배열 또는 객체
- **TXT**: 탭 구분 텍스트

#### ✅ 자동 검증
- 필수 필드 체크
- 데이터 타입 검증
- 범위 검증
- 실시간 미리보기

```

```

   😄 유머러스 - 재미있고 경쾌한 톤

✅ 추천 개수 (1~10개 슬라이더)
✅ AI 추가 지침 (특별 요청사항)
